<p align="center">냉장고의 식재료로 레시피를 추천받는 웹 서비스</p>

---

## 📌 프로젝트 소개

이 프로젝트는 사용자가 보유한 재료와 태그를 기반으로,  
사용자에게 어울리는 레시피를 추천해주는 웹 애플리케이션입니다.  
프론트엔드는 React, 백엔드는 Node.js/Express를 기반으로 제작되었습니다.  

재료 유사도 정렬, 레시피 API 연동, 사용자 편의성을 고려한 UI 등을 포함합니다.
